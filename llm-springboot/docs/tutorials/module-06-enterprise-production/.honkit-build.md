# HonKit Build Instructions

This file contains instructions for building the tutorial with HonKit.

## Prerequisites

- Node.js 16+ and npm installed
- Terminal access

## Installation

```bash
cd docs/tutorials/module-06-enterprise-production
npm install
```

This installs:
- `honkit` - Static site generator for tutorials
- `honkit-plugin-mermaid-hybrid` - Mermaid diagram rendering

## Build Commands

### Development Server (Live Reload)

```bash
npm run serve
```

Then open http://localhost:4000 in your browser.

The server watches for file changes and auto-reloads.

### Production Build

```bash
npm run build
```

Generates static HTML in `_book/` directory.

## Troubleshooting

### Issue: "Cannot find module 'honkit'"

**Solution**: Run `npm install` to install dependencies.

### Issue: "Mermaid diagrams not rendering"

**Solution**: Ensure `honkit-plugin-mermaid-hybrid` is installed:

```bash
npm install honkit-plugin-mermaid-hybrid
```

### Issue: "Port 4000 already in use"

**Solution**: Kill the process using port 4000:

```bash
# macOS/Linux
lsof -ti:4000 | xargs kill -9

# Or use a different port
honkit serve --port 4001
```

## Customization

### Modify Styling

Edit `styles/website.css` to customize appearance.

### Update Configuration

Edit `book.json` to change:
- Title and description
- Author information
- Plugin configuration
- Variables (module, difficulty, time)

## Publishing

### GitHub Pages

1. Build the tutorial:
   ```bash
   npm run build
   ```

2. Commit `_book/` directory (or configure CI/CD)

3. Configure GitHub Pages to serve from `_book/`

### Static Site Hosting

The `_book/` directory contains static HTML—deploy to:
- Netlify
- Vercel
- AWS S3 + CloudFront
- Any static hosting provider
