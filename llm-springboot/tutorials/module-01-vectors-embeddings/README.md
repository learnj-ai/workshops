# Building Semantic Vector Search with Spring Boot and Embeddings

Welcome to Module 01 of the LLM Spring Boot Workshop! This tutorial teaches you how to implement semantic vector search from scratch using Spring Boot and machine learning embeddings.

## What You'll Build

A complete vector search system that converts documents into numerical embeddings and enables similarity-based search. You'll learn the fundamental physics of AI-powered search and how to architect retrieval systems for enterprise applications.

## Technologies Used

- Java 17+
- Spring Boot 3.x
- LangChain4j
- AllMiniLmL6V2 Embedding Model
- Maven

## Getting Started

Follow the chapters in order to build your understanding from foundational concepts to the complete working system.

**Estimated Time**: 3-4 hours

**Difficulty**: Intermediate
