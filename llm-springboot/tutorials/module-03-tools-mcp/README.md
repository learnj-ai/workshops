# AI Tools & Model Context Protocol Integration

Welcome to Module 03 of the LLM Spring Boot Workshop! This tutorial teaches you how to build AI assistants that can autonomously use tools to access databases and external APIs.

## What You'll Build

A complete AI assistant with tool execution capabilities that can query PostgreSQL databases, call weather APIs, and synthesize results into natural language responses. You'll learn how function calling works and how to implement the Model Context Protocol (MCP) for tool orchestration.

## Technologies Used

- Java 17+
- Spring Boot 3.x
- LangChain4j
- OpenAI GPT-4o-mini
- PostgreSQL
- Maven

## Getting Started

Follow the chapters in order to build your understanding from foundational concepts to the complete working system.

**Estimated Time**: 3-4 hours

**Difficulty**: Intermediate

## Prerequisites

- Completion of Module 01 (recommended)
- OpenAI API key (sign up at platform.openai.com)
- PostgreSQL running locally or in Docker
