# LLM Spring Boot Workshop

Welcome to the comprehensive LLM Spring Boot Workshop! This tutorial will guide you through building production-ready LLM-powered applications using Spring Boot and modern AI techniques.

## What You'll Learn

This workshop is organized into six comprehensive modules, each focusing on a critical aspect of building LLM applications:

### Module 1: Vectors & Embeddings
Learn the fundamentals of vector search, embeddings, and semantic similarity. Build a complete vector store service from scratch.

### Module 2: Advanced RAG
Master advanced Retrieval-Augmented Generation techniques including query transformation, hybrid search, and re-ranking strategies.

### Module 3: Tools & MCP
Discover how to extend LLM capabilities with custom tools and the Model Context Protocol (MCP) for enhanced functionality.

### Module 4: From Chatbots to Agents
Transform simple chatbots into intelligent agents using the ReAct pattern, multi-agent orchestration, and conversation memory.

### Module 5: Security & Guardrails
Implement essential security measures including prompt injection protection, PII masking, output validation, and access control.

### Module 6: Enterprise & Production
Prepare your application for production with observability, caching, token optimization, and evaluation using Dokimos.

## Prerequisites

- Java 17 or higher
- Maven or Gradle
- Basic understanding of Spring Boot
- Familiarity with REST APIs
- Interest in LLM and AI applications

## How to Use This Tutorial

Each module is self-contained but builds upon concepts from previous modules. We recommend following them in order, but you can also jump to specific topics of interest.

Each chapter includes:
- Conceptual explanations
- Code examples with detailed annotations
- Best practices and common pitfalls
- Hands-on exercises

Navigate to **Module 1: Vectors & Embeddings** to begin your journey into building powerful LLM applications with Spring Boot!
