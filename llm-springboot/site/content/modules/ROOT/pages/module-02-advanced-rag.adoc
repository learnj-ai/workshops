== 3. Background

=== 3.1. Hybrid Search

Combining keyword search (BM25) with vector search using *Reciprocal Rank Fusion (RRF)* yields better recall than either method alone.

=== 3.2. Cross-Encoder Re-ranking

An initial retrieval pass returns candidates; a cross-encoder then scores each (query, document) pair for precise relevance.

=== 3.3. Query Transformation

* *Multi-Query:* Generate multiple perspectives of the same query to improve recall.
* *HyDE:* Generate a hypothetical answer document and use its embedding for retrieval.
